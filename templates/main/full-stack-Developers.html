Complete Course Syllabus

The Full Stack Web Development Cramdegree makes you a professional who is skilled to work with front-end and back-end web technologies. By developing proficiency in all these areas, a full stack web developer becomes a highly valuable professional and earns a higher salary than other kinds of developers. Therefore, full stack web developers are in great demand all over the world.

Complete Course Syllabus

Introduction to the course
HTML & CSS

HTML5, CSS, Responsive Websites using Media Queries and Bootstrap framework.

JavaScript & ReactJS

Basics of Programming Languages, JavaScript, Puzzles, Understanding the DOM, AJAX Calls and ReactJS.

Backend Programming

Node Js, Understanding HTTP and REST APIs. Writing your own REST APIs using Express. Connecting an Express Server with Mongo DB.

Database

Using MongoDB to store and read data. Writing queries to get data, using a Command Line to access Mongo DB. Creating a cluster and accessing it via a GUI (graphical user interface).

Version Control Software

Learn how to collaborate and version source code using Git and Github.
